#  Titanic Survival Prediction using Machine Learning

##  Project Overview
This project predicts Titanic passenger survival using multiple machine learning classification algorithms.  
It analyzes passenger features such as age, sex, fare, and class to determine survival probability.

---

##  Features
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature encoding and scaling
- Train-test split
- Machine learning models:
  - Logistic Regression
  - Decision Tree Classifier
  - Random Forest Classifier
  - Support Vector Machine (SVM)
  - K-Nearest Neighbors (KNN)
  - Naive Bayes
- Model evaluation (Accuracy, Confusion Matrix, Classification Report)
- Model comparison

---

##  Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

##  Dataset
- Titanic dataset (1309 passengers)
- Features:
  - Age
  - Fare
  - Sex
  - Pclass
  - SibSp
  - Parch
  - Embarked
- Target:
  - Survived (0 = No, 1 = Yes)

---

##  Models Used

###  Logistic Regression
- Baseline classification model

###  Decision Tree
- Handles non-linear patterns

###  Random Forest
- Ensemble model for better accuracy

###  SVM
- Strong performance on classification tasks

###  KNN
- Distance-based classifier

###  Naive Bayes
- Probabilistic model

---

##  Results (Approx.)
- Logistic Regression: ~77%
- Decision Tree: ~71%
- Random Forest: ~71%
- SVM: ~79% (Best Model)
- KNN: ~73%
- Naive Bayes: ~72%
